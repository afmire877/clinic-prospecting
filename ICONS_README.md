# Icons Setup

The extension requires three icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels) 
- icon128.png (128x128 pixels)

You can create these icons manually or use any icon generator. The icons should represent the extension's purpose (copying markdown).

For now, you can use any 16x16, 48x48, and 128x128 PNG files as placeholders, or remove the icons section from manifest.json if you prefer to run without icons initially.
